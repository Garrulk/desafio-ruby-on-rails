

# Desafio programação - para vaga desenvolvedor Ruby on Rails

# Versões Utilizadas

* Ruby 2.7.0
* Rails 6.0.3

# Funcionalidades da aplicação

* Tela inicial com um campo para a importação de um arquivo txt (CNAB.txt)
* Ao ser importado, será mostrada uma tabela com os dados que se encontram dentro do arquivo, já parseados
* Aparecerá também um link para a a visualização do saldo separado por nome da loja

# Gems Utilizadas 

* Devise
* Rubocop
* Rspec-rails
* pg
* FFaker
* FactoryBot

# Para rodar a aplicação em localhost

* Fazer o clone da aplicação em https://github.com/Garrulk/desafio-ruby-on-rails.git
* Criar o Banco de Dados
* Fazer as migrações
* Abrir o browser no endereço localhost:3000
* Criar um usuário com e-mail e senha
* Fazer o login na aplicação
* Importar o arquivo CNAB
